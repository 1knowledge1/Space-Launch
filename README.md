# Space Launch
# Техническое задание

## Назначение
Приложение, в котором можно получить актуальную информацию о предстоящих и прошедших космических запусках. Самое интересное о космосе в одном месте!

## Целевая аудитория
Люди, увлекающиеся космосом, а также те, кто интересуется космическими запусками и не хочет пропустить запуск ракета-носителей.

## Функциональность приложения
1. Вкладка "Запуски":
    - Добавить возможность просмотра предстоящих и прошедших запусков ракет (используется API [Launch Library 2](https://thespacedevs.com/llapi));
    - Добавить возможность настроить уведомления об интересующих запусках в ленте "Предстоящие";
    - Просмотр информации о запусках по нажатию на элемент;
    - Добавить cсылки на стримы на экран с подробной информацией при переходе из списка "Предстоящие";
    - Показывать результат запуска на экране с подробной информацией при переходе из списка "Прошедшие".
2. Вкладка "Космофото":
    - Возможность просмотра ежедневной фотографии космоса от NASA (используется API [NASA APOD](https://api.nasa.gov/)).
3. Вкладка "Ракеты":
    - Возможность просмотра различных моделей ракет и подробной информации о них;
    - Организовать поиск.
4.  Вкладка "Тесты":
    - Возможность просмотра и выбора теста на космическую тематику;
    - Возможность прохождения теста с сохранением результатов;
    - Возможность поделиться результатами в соцсети "ВКонтакте".
